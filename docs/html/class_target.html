Target Class Reference

The Target class has composition of all other classes for triggering the whole project. Specifically, it creates two copies of the Maze class, for the two robots (Wheeled and Tracked). It then takes in user input for locations of robots and targets. Later it triggers the A* algorithm from the Maze class to find out the optimal paths (if they exist).
